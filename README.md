# YESNOWTF

Get a random gif of approval or disapproval, powered by the yesno.wtf API.

## USAGE

``` sh
git clone https://github.com/vzvenyach/yesnowtf.git
cd yesnowtf
./run.sh
```

Or better yet....

`curl https://raw.githubusercontent.com/vzvenyach/yesnowtf/master/run.sh | bash`
